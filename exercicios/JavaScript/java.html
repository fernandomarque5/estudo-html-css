Inserindo botoes interativos

para inserir botões em HTML, utiliza-se a TAG
"button", veja o botão abaixo

para atribuirmos uma interação com o botão, precisamos utilizar o JavaSript, e criar uma função ao qual queremos que aconteça quando clicarmos no botão
Ex: quando clicar no botão, irá aparecer um alerta na tela, indicando que voce apertou o botão

Podemos também utilizar o JavaSript, para quando clicar em um botão, ele mude a cor de fundo do site

Podemos tambem criar eventos com interação de caixas.
ex: ao passar o ponteiro do mouse na caixa, ela mude o conteúdo escrito. ao sair e clicar também
